CTF Tools

This repository contains a suite of tools I have built in an attempt to understand common tools such as port scanners, repeaters, and password crackers - see it on Github here

It is a constant work in progress - as you can see by the issues board!

I also intend to use these tools as a teaching resource for Sheffield Ethical Student Hackers Society - automation is a fundamental tool, so this repo will hopefully show some useful examples of how to apply these skills to CTFs and boxes
